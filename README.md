# mesas-examen
- __Frontend__: Vue
- __Backend__: FastAPI
## Iniciar Backend
> Ubicación
```bash
cd mesas-examen
```
> Crear entorno virtual
```bash
python -m venv venv
```
> Activar entorno virtual
```bash
venv\scripts\activate
```
> Instalar dependencias
```bash
pip install -r requirements.txt
```
> Ubicación
```bash
cd backend
```
> Ejecutar el backend
```bash
fastapi dev app/main.py
```
## Iniciar Frontend
> Ubicación
```bash
cd frontend
```
> Instalar dependencias
```bash
npm install
```
> Ejecutar el frontend
```bash
npm run dev
```
